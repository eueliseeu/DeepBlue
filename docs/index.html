Por que usar o DeepBlue?

Focado em produtividade para desenvolvedores modernos. Suporte a 7 linguagens.

Detecção Automática

Identifica automaticamente sua linguagem e versão instalada. Zero config, máxima produtividade.

7 Linguagens Suportadas

Node.js, Python, Go, Java, PHP, Ruby, Rust. Templates otimizados e atualizados para produção.

.dockerignore Automático

Gera arquivos .dockerignore otimizados por linguagem, reduzindo o build context e acelerando builds.

1 Instalação

Você pode instalar o DeepBlue globalmente usando o NPM.

npm install -g deepblue-cli

2 Guia Rápido

Inicializar ambiente Docker

Inicia um wizard interativo para configurar Dockerfile e docker-compose.yml.

deepblue init

Verificar versão

Exibe a versão atual do DeepBlue CLI instalada.

deepblue --version
